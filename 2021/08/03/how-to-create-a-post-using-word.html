This is a minimum example on creating a post entry using Microsoft Word. Please notice that when using a Word document there is less flexibility than when using a Markdown document or a Jupyter Notebook.

  1. Please remember to use the name convention for all posts. This means you Word document should be named:

YYYY-MM-DD-title-of-your-post.docx

  1. You can include titles and links in your document.
